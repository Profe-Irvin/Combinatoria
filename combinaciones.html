Palabras Clave en Combinatoria

Tipo Palabras Clave y Ejemplos Características
Permutación Sin Repetición Arreglar, Ordenar, Colocar, Disposición, Secuencia, Rotación, Clasificación, Organizar, Distribución
Ejemplos:
  • ¿De cuántas maneras se pueden arreglar 6 fotos en un álbum?
  • Si tienes 5 libros, ¿de cuántas maneras puedes ordenar los libros en una estantería?
 No se pueden repetir, todos los elementos pueden ser usados, el orden influye.
Permutación Con Repetición Arreglar, Ordenar, Claves, Contraseñas, Secuencia, Repetición, Formar, Organizar
Ejemplos:
  • ¿Cuántas contraseñas de 4 dígitos se pueden formar usando los números del 0 al 9?
  • Si tienes 3 letras (A, B, C), ¿cuántas secuencias de 3 letras se pueden formar?
 Se pueden repetir, todos los elementos pueden ser usados, el orden influye.
Combinación Sin Repetición Elegir, Seleccionar, Grupo, Subgrupo, Sin importar el orden, Recoger, Formar, Agrupar, Reunir
Ejemplos:
  • ¿Cuántos grupos de 3 frutas se pueden elegir de una canasta de 10?
  • Si hay 12 estudiantes, ¿de cuántas maneras se pueden formar equipos de 4?
 No se pueden repetir, todos los elementos pueden ser usados, el orden no influye.
Combinación Con Repetición Elegir, Seleccionar, Grupo, Reunir, Sin importar el orden, Claves, Variantes, Repetición
Ejemplos:
  • ¿Cuántas formas hay de elegir 3 frutas de una canasta con 5 tipos, permitiendo repetir?
  • ¿Cuántos subgrupos de 2 personas se pueden crear si cada persona puede ser seleccionada varias veces?
 Se pueden repetir, todos los elementos pueden ser usados, el orden no influye.
Variación Sin Repetición Seleccionar con orden, Elegir con orden, Arreglar, Asignar, Clasificar, Distribuir, Organizar
Ejemplos:
  • ¿De cuántas maneras se pueden asignar 3 puestos de trabajo a 5 candidatos?
  • ¿Cuántas secuencias se pueden formar con 4 números tomados de un conjunto de 6?
 No se pueden repetir, no todos los elementos pueden ser usados, el orden influye.
Variación Con Repetición Seleccionar con orden, Elegir con repetición, Claves, Contraseñas, Secuencias, Formar, Organizar
Ejemplos:
  • ¿Cuántas combinaciones de 4 letras se pueden formar usando A, B y C con repetición?
  • Si hay 3 sabores de helado, ¿cuántas variantes se pueden hacer al elegir 2, si el orden importa y se pueden repetir?
 Se pueden repetir, no todos los elementos pueden ser usados, el orden influye.
